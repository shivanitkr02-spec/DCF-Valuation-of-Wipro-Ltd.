# DCF Valuation of Wipro Limited

## Overview
This project presents a Discounted Cash Flow (DCF) valuation of Wipro Ltd using the FCFF approach.

## Key Features
- 5-year financial projections (FY25–FY29)
- WACC estimation
- Terminal value calculation
- Sensitivity analysis (WACC & growth rate)

## Tools Used
- Microsoft Excel

## Files Included
- Wipro_DCF_Model.xlsx (Financial Model)
- Wipro_DCF_Valuation_Report.pdf (Detailed Report)

## Outcome
Estimated intrinsic value and analyzed key financial drivers and valuation sensitivity.
